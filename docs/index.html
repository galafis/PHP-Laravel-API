API de Gerenciamento de Tarefas

Bem-vindo à documentação da API de Gerenciamento de Tarefas. Esta API RESTful permite criar, ler, atualizar e excluir tarefas.

Endpoints

Listar todas as tarefas

GET /api/tasks

Criar uma nova tarefa

POST /api/tasks
Content-Type: application/json

{
    "title": "Minha Nova Tarefa",
    "description": "Descrição detalhada da tarefa.",
    "completed": false
}

Obter uma tarefa específica

GET /api/tasks/{id}

Atualizar uma tarefa existente

PUT /api/tasks/{id}
Content-Type: application/json

{
    "title": "Tarefa Atualizada",
    "completed": true
}

Excluir uma tarefa

DELETE /api/tasks/{id}
